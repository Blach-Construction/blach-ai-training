# Blach AI Training Hub

An internal, employee-facing training hub for getting started with Claude at Blach Construction.

**Live site:** https://blach-construction.github.io/blach-ai-training/ (GitHub Pages, internal/private repo)

## What it is
- **Watch & Learn** — six curated YouTube walkthroughs, rendered inline (privacy-enhanced embeds).
- **The Toolkit** — the 7 Claude tools (Chat, Projects, Skills, Connectors, Artifacts, Cowork, Claude Code).
- **Guides** — downloadable one-page PDF quick-start guides.
- **Progress tracking** — saved per-device in the browser (localStorage). No accounts, no server.
- Blach-branded design, animations, and confetti.

## Tech
Pure static site — HTML/CSS/vanilla JS, no build step. Hosted on GitHub Pages.

## Editing content
Videos and tool cards are plain HTML in `index.html`. Brand tokens live in `css/brand.css`.
