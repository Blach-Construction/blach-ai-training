The Toolkit

Learn by doing

Claude is one platform with a handful of tools — each suited to a different kind of work. Tap a card to mark it explored as you try each one.

Claude (Chat)

The everyday assistant. Ask questions, draft and edit writing, summarize long documents or analyze information through a simple conversation.

Use it whenyou need a fast first draft, a quick summary or a clear answer. ✦ Tap to mark explored

Projects

A dedicated workspace that holds your instructions and reference files so every chat starts with the right context.

Use it whenyou do recurring work like proposals and want Claude primed every time. ✦ Tap to mark explored

Skills

Reusable instructions that teach Claude to do a task the same way each time — like our Blach branding skill.

Use it whenyou want on-brand, consistent output without re-explaining the rules. ✦ Tap to mark explored

Connectors

Secure links between Claude and the apps we already use, such as Egnyte, Box, Monday.com and Microsoft 365.

Use it whenyou want Claude to pull from or work with company files and tools. ✦ Tap to mark explored

Artifacts

Editable documents, slides, spreadsheets and visuals that build right beside your chat.

Use it whenyou are creating a deliverable you will refine, reuse and download. ✦ Tap to mark explored

Cowork

A desktop app where Claude carries out multi-step tasks on its own across your files and connected apps.

Use it whena job has several steps and you want Claude to run the whole sequence. ✦ Tap to mark explored

Claude Code

An agentic tool for building and automating — from simple scripts to working tools, sites and apps. No coding background required.

Use it whenyou want to automate a workflow or build a small custom tool. ✦ Tap to mark explored

New here? Start simple ✨

Open Chat and ask Claude to help with a real task you have today — a summary, an email or a first draft. Add Projects, Skills and Connectors as your work grows.
